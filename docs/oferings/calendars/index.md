# Calendários

Nesta pagina voce cria e mantem calendarios academicos usados nas alocacoes.

## O que aparece na tela

- Lista de calendarios ativos e inativos
- Periodos, unidades e status

## Acoes comuns

- Criar calendario para um periodo
- Definir datas de inicio e fim
- Atualizar feriados e eventos

## Categoria de Feriados

Use as categorias para organizar feriados (ex.: nacional, estadual, institucional).
