# Como usar o USPolis?

A primeira coisa que você deve fazer é se **registrar no sistema**, através da barra de navegação no botão "Entrar", a seguir a imagem da barra de navegação que aparece na página inicial.

Ao clicar no botão você verá a seguinte tela, utilize a sua conta **usp**, caso contrário não irá conseguir se registrar.

      height: auto;

  />
</div>

Pronto, agora você tem uma conta no USPolis! Agora você será redirecionado para a tela do [Mapa de Salas](/allocation). No canto superior esquero você encontra um botão para abrir o menu lateral, nele você pode encontrar todas as funcionalidades disponíveis para o seu usuário.
