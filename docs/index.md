# Bem-vindo à documentação do USPolis

Aqui você vai encontrar uma explicação detalhada sobre todas as funcionalidades do USPolis e qualquer outra dúvida que possa surgir.

Essa documentação é divida por páginas, cada seção no menu lateral esquerdo contém uma explicação de como utilizar aquela página em específico. Se você for um **usuário comum** as páginas de seu interesse são: 
- [Mapa de Salas](./allocation/)
- [Solicitar uma Sala](./solicitate-a-classroom/)
- [Encontre suas provas](./find-exams/)
- [Encontre suas aulas](./find-classes/)
- [Minhas solicitações](./my-solicitations/)

Cada seção contem um ou mais itens com explicações, imagens e possívelmente vídeos mostrando como usar.

> [!IMPORTANT]
> **A documentação está em desenvolvimento**. Não encontrou uma explicação para a sua dúvida? Fale conosco através de **uspolis@usp.br**

> [!TIP]
> A documentação técnica pode ser encontrada no botão [Referencia](./dev/), no menu superior. Um [Changelog](./changelog/) pode ser encontrado também no menu superior.
