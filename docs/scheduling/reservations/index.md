# Reservas

Esta pagina lista as reservas aprovadas e permite acompanhamento por sala e periodo.

## O que aparece na tela

- Lista de reservas com data, horario, sala e solicitante
- Status da reserva (ativa, cancelada, concluida)
- Filtros por periodo, sala e tipo de reserva

## Acoes comuns

- Criar uma reserva (quando permitido)
- Editar horario, sala ou detalhes
- Cancelar uma reserva
