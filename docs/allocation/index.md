# Mapa de Salas

O mapa de salas é a principal página do USPolis, é nele onde você consegue encontrar todas as disciplinas e reservas disponíveis. É através dessa página que você pode fazer solicitações de sala (veja detalhes em [Solicitar uma sala](../solicitate-a-classroom/)).

O botão "Baixar" serve para baixar diversos tipos de relatórios em formato PDF, para mais detalhes acesse [Relatórios](./pdfs). O segundo serve para solicitar uma reserva de sala, é necessário que você esteja logado no sistema, use seu email usp, caso contrário não irá conseguir se autenticar.

Os seletores servem para você escolher ver dados de um prédio em específico, uma sala em específico (é necessário escolher um prédio primeiro), uma disciplina em específico e uma turma em específico (é necessário escolher uma disciplina primeiro). Não existe um seletor para reservas.

No canto esquerdo é possível encontrar o seguinte menu (essas opções são para usuários comuns, demais usuários tem mais opções):

Uma outra forma de **avançar** ou **voltar** nas datas é utilizando os **botões** que estão no lado direito do cabeçalho do calendário, as duas setas (< >), conforme a imagem a seguir:
