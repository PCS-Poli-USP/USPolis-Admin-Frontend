## Como ver provas antigas

É possível procurar por provas antigas usando o filtro que aparece ao lado do título da página, ao clicar nele você verá:

<div style="
  display: flex;
  justify-content: center;
  align-items: center;
  width: 100%;
">
  <img 
    src="/find-exams/image-5.png"
    alt="Descrição"
    style="
      border-radius: 8px;
      max-width: 100%;
      height: auto;
      box-shadow: 0 4px 16px rgba(0,0,0,0.1);
    "
  />
</div>

Basta inserir o período que deseja buscar, esse período deve conter **todas as datas de uma reserva do tipo "Prova"**, por exemplo, suponha que para MAC2166 existia uma reserva do tipo "Provas" com as datas: 24/04/25 (P1) e 26/06/25 (P2) então o seu intervalo deve contem ambas as datas para que ela apareça no site.
