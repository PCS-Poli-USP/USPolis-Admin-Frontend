# O que é o USPolis?

O USPolis é um sistema de código aberto para a alocação e gestão de salas de aula. Uma vez que a alocação é feita, a informação é distribuida através do site [USPolis](https://uspolis.com.br/) e dos aplicativos para [Android](https://play.google.com/store/apps/details?id=uspolis.lunadros) e [iOS](https://apps.apple.com/br/app/uspolis/id1451455075).

Atualmente o USPolis é utilizado na Escola Politénica da Universidade de São Paulo (Poli-USP), é desenvolvido e mantido pelos alunos da USP e feito com o objetivo de ajudar os alunos a se localizarem, organizarem e descobrirem onde são suas aulas.
