# Administração

Esta área concentra ferramentas administrativas para usuarios com permissao de administrador.

## O que voce pode fazer

- Gerenciar usuarios e grupos
- Acompanhar sessoes e acessos
- Configurar eventos institucionais e cursos
- Revisar bug reports e feedbacks

## Acesso

Somente perfis com permissao de administrador conseguem ver este menu.
